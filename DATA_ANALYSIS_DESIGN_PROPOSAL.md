## 📌 개요

본 문서는 아파트 분양 사전영업 설문 데이터를 **전문가 수준**으로 분석하기 위한 종합 설계 제안서입니다. 단순 통계를 넘어, 고객 행동 예측, 세그먼트 기반 타겟팅, 주차별 트렌드 분석 등 **실무에서 즉시 활용 가능한 인사이트**를 도출하는 것을 목표로 합니다.

## 🎯 1. 분석 목표 및 KPI 체계

### 1.1 핵심 분석 목표

| 목표 | 설명 | 기대 효과 |
|------|------|-----------|
| **고객 품질 평가** | S/A/B/C 등급별 고객 특성 파악 | 핵심 타겟 집중 공략 |
| **유입 채널 효과 분석** | 채널별 전환율 및 고객 품질 비교 | 마케팅 예산 최적화 |
| **수요 예측** | 평형별, 가격대별 선호도 분석 | 상품 기획 및 가격 전략 수립 |
| **시계열 트렌드 파악** | 주차별 변화 추이로 캠페인 효과 측정 | 실시간 전략 조정 |

## 👥 2. 고객 세분화 (Segmentation) 전략

### 2.1 다차원 세분화 모델

#### A. 인구통계학적 세분화 (Demographic)

| 변수 | 분류 기준 | 마케팅 시사점 |
|------|-----------|---------------|
| **연령대** | 20대 / 30대 / 40대 / 50대+ | 디지털 채널 vs 전통 채널 선호도 |
| **성별** | 남성 / 여성 | 커뮤니케이션 톤앤매너 조정 |
| **거주지** | 사업지 인근 / 동일 광역시 / 타지역 | 실거주 vs 투자 목적 예측 |

#### B. 행동 기반 세분화 (Behavioral)

| 세그먼트 | 정의 | 우선순위 |
|----------|------|----------|
| **🔥 Hot Lead** | 의향 7점 + 청약자격 O + 예산 적합 | ⭐⭐⭐⭐⭐ |
| **🌟 Warm Lead** | 의향 5~6점 + 청약자격 O | ⭐⭐⭐⭐ |
| **💡 Potential** | 의향 5점 이상 + 청약자격 X (추후 확보 가능) | ⭐⭐⭐ |
| **❄️ Cold Lead** | 의향 4점 이하 | ⭐ |

#### C. 목적 기반 세분화 (Psychographic)

```python
# 구매 목적(Q4)에 따른 세그먼트
실거주형 = "실거주" 선택 고객  # 안정적 계약, 장기 거주
투자형 = "투자" 선택 고객      # 수익률 중시, 가격 민감
혼합형 = "실거주+투자" 선택   # 복합 니즈, 프리미엄 관심
```

### 2.2 세그먼트별 맞춤 전략

| 세그먼트 | 핵심 메시지 | 채널 전략 | 콘텐츠 유형 |
|----------|-------------|-----------|-------------|
| Hot Lead | "지금 잡으세요" | 1:1 전화, 방문 유도 | VIP 혜택, 잔여호수 안내 |
| Warm Lead | "이런 혜택이!" | SMS, 카카오톡 | 할인/프로모션, 모델하우스 초대 |
| Potential | "청약 준비하세요" | 이메일, 블로그 | 청약 가이드, 청약통장 안내 |
| Cold Lead | "관심 가져주세요" | 리타겟팅 광고 | 브랜드 스토리, 입지 정보 |

## 📈 3. 고급 분석 기법

### 3.1 RFM 분석 (Recency-Frequency-Monetary 변형)

사전영업 특성에 맞게 **RFIE 모델**로 변형 적용:

| 지표 | 원래 의미 | 사전영업 적용 | 점수 산정 |
|------|-----------|---------------|-----------|
| **R** (Recency) | 최근 구매일 | 최근 응답/접촉일 | 최근일수록 높음 |
| **F** (Frequency) | 구매 빈도 | 접촉/문의 횟수 | 많을수록 높음 |
| **I** (Intent) | 금액 | 계약 의향 점수 (Q6) | 높을수록 높음 |
| **E** (Eligibility) | - | 청약 자격 보유 (Q7) | 보유시 +가점 |

```
RFIE Score = R(1-5) + F(1-5) + I(1-5) + E(0 or 2)
최대 점수: 17점
```

**세그먼트 분류:**
- 🏆 **Champions (15-17)**: 즉시 계약 가능 고객
- ⭐ **Loyal (12-14)**: 높은 관심, 추가 설득 필요
- 🌱 **Promising (8-11)**: 육성 대상
- 💤 **At Risk (5-7)**: 관심 저하, 재활성화 필요
- ❌ **Lost (1-4)**: 장기 육성 또는 제외

### 3.2 코호트 분석 (Cohort Analysis)

```
┌──────────────────────────────────────────────────────────┐
│                    주차별 코호트 분석                      │
├──────────────────────────────────────────────────────────┤
│                    가입 후 경과 주차                       │
│  가입 주차  │  0주  │  1주  │  2주  │  3주  │  4주      │
├──────────────────────────────────────────────────────────┤
│  1주차 코호트 │ 100% │  85% │  72% │  68% │  65%      │
│  2주차 코호트 │ 100% │  90% │  78% │  70% │   -       │
│  3주차 코호트 │ 100% │  88% │  75% │   -  │   -       │
│  4주차 코호트 │ 100% │  82% │   -  │   -  │   -       │
└──────────────────────────────────────────────────────────┘
* 비율 = 각 주차에 여전히 "고관심(S급)" 상태인 비율
```

**분석 포인트:**
- 1주차 → 2주차 이탈률이 높다면? → 초기 팔로업 강화 필요
- 특정 코호트의 잔존율이 높다면? → 해당 기간 마케팅 전략 분석

### 3.3 리드 스코어링 (Lead Scoring)

#### 스코어링 기준표

| 항목 | 기준 | 점수 |
|------|------|------|
| **계약 의향 (Q6)** | 7점 | +30 |
| | 5~6점 | +20 |
| | 3~4점 | +10 |
| | 1~2점 | +0 |
| **청약 자격 (Q7)** | 보유 | +25 |
| | 미보유 | +0 |
| **구매 목적 (Q4)** | 실거주 | +15 |
| | 투자 | +10 |
| | 기타 | +5 |
| **희망 분양가 (Q8)** | 실제 분양가 범위 내 | +20 |
| | 범위 외 | +0 |
| **유입 채널 (Q2)** | 지인 추천 | +15 |
| | 현장 방문 | +12 |
| | 온라인 광고 | +8 |
| | 기타 | +5 |

**등급 분류:**
- 🔴 **A급 (80점 이상)**: 즉시 전화 상담
- 🟠 **B급 (60-79점)**: 48시간 내 문자/카톡
- 🟡 **C급 (40-59점)**: 주간 뉴스레터
- ⚪ **D급 (40점 미만)**: 월간 리마인드

## 📊 4. 시계열 트렌드 분석

### 4.1 주차별 핵심 지표 추적

| 지표 | 분석 목적 | 경고 기준 |
|------|-----------|-----------|
| 신규 응답 수 | 마케팅 도달력 | 전주 대비 -20% 이상 하락 |
| 평균 의향 점수 | 고객 품질 변화 | 전주 대비 -0.5점 이상 하락 |
| S급 비율 | 핫리드 유입률 | 전주 대비 -5%p 이상 하락 |
| 채널별 비중 | 채널 효과 변화 | 특정 채널 급증/급감 |

### 4.2 조기 경보 시스템

```
⚠️ 경보 조건 예시:
- 2주 연속 응답 수 10% 이상 감소 → 광고 집행량/채널 점검
- 평균 의향 점수 5.0 이하 하락 → 타겟팅 정확도 점검
- 특정 평형 선호도 급등 → 재고 확인 및 홍보 강화
```

## 🤖 5. AI 인사이트 고도화 방안

### 5.1 현재 AI 분석 범위

```
현재 구현: 
├─ 전체 통계 요약
├─ 핵심 타겟 분석
├─ 주차별 트렌드 분석
└─ 마케팅 제언 1~2가지
```

### 5.2 고도화 제안

| 기능 | 설명 | 기대 효과 |
|------|------|-----------|
| **세그먼트별 분석** | RFM/리드스코어 기반 그룹별 인사이트 | 맞춤 전략 수립 |
| **이상치 탐지** | 갑작스러운 지표 변화 감지 | 즉각 대응 가능 |
| **예측 분석** | 향후 2주 응답 수/S급 비율 예측 | 선제적 의사결정 |
| **경쟁사 비교** | 유사 사업지 대비 성과 벤치마킹 | 시장 포지셔닝 |
| **자연어 질의** | "마포구 고객 특성은?" 형태 질문 | 사용 편의성 향상 |

### 5.3 AI 프롬프트 고도화 예시

```python
# 고도화된 프롬프트 구조
prompt = f"""
당신은 10년 경력의 아파트 분양 마케팅 전문 컨설턴트입니다.

# 역할
- 데이터 기반 의사결정 전문가
- 고객 행동 분석 전문가
- 분양 마케팅 트렌드 전문가

# 분석 데이터
{전체_통계}
{주차별_트렌드}
{세그먼트별_분포}
{RFM_분석_결과}

# 분석 요청
1. 가장 주목해야 할 데이터 패턴 3가지
2. 즉시 실행 가능한 액션 아이템 3가지
3. 주의해야 할 리스크 요인 2가지
4. 다음 주 예상 시나리오 (낙관/중립/비관)

# 출력 형식
마크다운 형식으로, 경영진 보고용 간결체로 작성
"""
```
